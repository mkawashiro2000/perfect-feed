Support Vector Machines: A Progression of Algorithms

El modelo de aprendizaje Machine Learning (ML) que se basa en vectores de soporte es un algoritmo popular utilizado para resolver problemas de clasificación. Es conocido por su capacidad de funcionar sin mucha configuración manual y es valioso para aplicaciones donde el conocimiento de las fronteras de las clases es más importante que el conocimiento de las distribuciones de clase. Sin embargo, cuando se trabaja con este tipo de modelos, se pueden mencionar otros algoritmos relacionados como Clasificador de Vectors de Soporte (SVC) o Clasificador de Margen Máximo (MMC). Aunque estos algoritmas están relacionados, es importante destacar la distinción entre ellos. Para comprender completamente SVMs, es necesario apreciar el progreso de los siguientes algoritmos, ordenados desde la menor a la mayor complejidad: MMC -> SVC -> SVM En este progreso, cada algoritmo extiende su funcionalidad para encontrar una frontera de decisión cada vez más flexible y robusta. Entender este progreso nos permite apreciar plenamente el poder de los modelos de vectores de soporte.
